 Portafolio Responsive con Vue.js

Este proyecto es un **portafolio web moderno y adaptable** desarrollado con HTML, CSS y Vue.js.  
Cuenta con animaciones suaves, una navegación fluida entre secciones y un menú hamburguesa totalmente funcional.

 Características principales

-  Diseño moderno con tipografía estilo developer (`Fira Code`).
-  Diseño 100% responsive adaptable a dispositivos móviles, tabletas y escritorio.
-  Menú hamburguesa interactivo con animación y cierre automático al seleccionar una opción.
-  Navegación interna fluida entre secciones (Inicio, Trabajos, Sobre mí, Contáctame).
-  Construido con Vue.js permitiendo componentes dinámicos y fácil manejo del DOM.
-  Fácil de personaliza y extender con nuevos estilos o secciones.

 Tecnologías utilizadas

- HTML5
- CSS3
- Vue.js (CDN o CLI)
- JavaScript
